Полноценное веб-приложение блога на Flask.

## Функционал
- Регистрация и авторизация пользователей
- CRUD статей (создание, чтение, редактирование, удаление)
- Иерархические комментарии (ответы на комментарии)
- Поиск по заголовкам статей
- Пагинация
- REST API (JSON)
- Валидация форм и обработка ошибок

## Стек
- Python
- Flask
- Flask-Login
- Flask-SQLAlchemy
- SQLite
- HTML / Jinja2 / CSS

## API
`GET /api/posts`

Ответ:
```json
[
  {
